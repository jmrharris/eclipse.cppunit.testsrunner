Eclipse.cppunit.testsrunner

CppUnit XML Report processing for the standard Eclipse CDT C/C++ Unit Testing Support plugin.

Welcome to Eclipse CppUnit TestsRunner

This Eclipse plugin integrates CppUnit C/C++ Unit Testing library XML Test Report processing into the Eclipse CDT "C/C++ Unit Testing Support" feature.

How it use the plugin

  1. Ensure you have installed the official Eclipse CDT "C/C++ Unit Testing Support" plugin.

  2. In the Eclipse "Project Explorer" right-click your C/C++ Project and select "Run As"->"Run Configurations..."

  3. Select the "C/C++ Unit" item from the tree view.

  4. Create a new "C/C++ Unit" launch configuration for your project.

  5. In the "C/C++ Unit Testing" tab, choose "CppUnit Tests Runner" from "Test Runner" drop down menu.

  6. Select "Apply" apply and then "Run"

  7. Your CppUnit XML Test Report results will now appear in the Eclipse "C/C++ Unit" view.

How it works

  1. Eclipse passes your test application's standard output to the CppUnit TestsRunner plugin.

  2. The plugin filters the output until it finds the character sequence '<', '?', 'x', 'm', 'l'.

  3. The plugin then treats the remaining test application output as a CppUnit default XML Test Report.

  4. The plugin adds each test result contained in the CppUnit XML Test Report to the "C/C++ Unit" view.
